# Changelog

## 0.1.0

- Initial TypeSea-powered valid sample generation.
- Invalid fuzz generation with user targets and built-in payload packs.
- Inspection API for caller-supplied payloads.
- Suite runner with timeout, retry, concurrency, failure predicates, and shrinking.
- Replay support for SeaFlow case ids.
- JSON-safe corpus persistence.
- TypeSea schema fallback with optional and undefinedable presence semantics.
